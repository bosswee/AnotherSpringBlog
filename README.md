## 开发

本地安装：

- [MySQL](https://www.mysql.com)
- [Redis](http://redis.io)
- [Pygments](http://pygments.org)

按需配置 `src/main/resources/application.yml` 

本地启动mysql,redis

命令行启动 ./gradlew bootRun
